US Quality Core adopts the [US Core Individual Provenance guidance](https://hl7.org/fhir/us/core/basic-provenance.html). Implementers should use the existing US Core individual-level data mappings as described where they exist. US Quality Core does not replace or modify those mappings.

US Quality Core extends the same individual-level provenance pattern to the US Quality Core profiles and profile variants that are not covered by the US Core guidance table. For each row below, the **Data Element Corresponding to Author Provenance** identifies the resource element that conveys the Author and the **Target Resource Types Corresponding to Author Role Provenance** identifies the referenced resources that convey the Author Role. This maps to the Draft USCDI+ Quality V2 Data Elements Author and Author Role in the Provenance Data Class.  

The mappings below apply in addition to the [US Core Individual Provenance guidance](https://hl7.org/fhir/us/core/basic-provenance.html).
