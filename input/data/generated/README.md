# Generated View Data

This directory contains committed JSON files generated by the project-specific
view-data generator.

These files are consumed by Liquid/Jekyll includes during the IG Publisher
build. Do not edit them by hand. To change this data, update the maintained
source JSON in `data/`, authored FSH under `input/fsh/`, or the generator logic
in `scripts/generate_view_data.js`, then rerun the generator.

## Files

- `profile_notes.json` - per-profile data used by profile intro and notes
  includes, including USCDI+ Quality flagged elements, US Core lineage facts,
  and resource-level search parameter expectations.
- `profile_table.json` - resolved data for the generated profile table on
  `input/pages/profiles.md`.
- `uscdi_quality_scope.json` - resolved in-scope and out-of-scope USCDI+
  Quality data element tables and profile lists for
  `input/pages/uscdiquality.md`.

The IG Publisher consumes these files through `site.data.generated.*`; it does
not regenerate them.
